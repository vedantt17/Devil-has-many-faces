# Devil Has Many Faces

A declassified document intelligence platform for exploring released government files — JFK, Epstein, and UAP/UFO disclosures.

## Features
- Full-text document search and browser
- Entity profiles with cross-document relationship mapping
- Chronological timeline view
- Ask the Files, RAG-powered chat grounded in source documents
- What's Still Hidden, redaction analysis and visualization
- Changelog feed for new document releases

## Tech Stack
- **Backend:** FastAPI, SQLite (FTS5), MongoDB, ChromaDB
- **AI:** sentence-transformers, Claude API (RAG)
- **Pipeline:** PyMuPDF, spaCy
- **Frontend:** React, Tailwind CSS
